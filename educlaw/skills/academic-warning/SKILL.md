# 学业预警

## 概述

基于预设规则检查学生学业状态，按严重程度生成分级预警报告，并给出补救措施建议。

## 触发关键词

预警、挂科、警告、风险、学分、不合格

## 预警规则

| 规则 | 条件 | 级别 |
|------|------|------|
| 课程挂科 | 有不及格课程 | critical |
| GPA < 2.0 | GPA低于2.0 | danger |
| GPA < 2.5 | GPA低于2.5 | warning |
| 学分缺口 | 缺口 > 10分 | warning |
| 能力下降 | 得分下降 | warning |

## 执行流程

1. 读取学生模拟数据（GPA 3.12, 20学分 / 140要求）
2. 逐条检查5项预警规则
3. 按严重程度排序：critical > danger > warning > info
4. 输出分级预警报告 + 行动建议

## 输出格式

预警报告：学生信息、GPA、各级预警消息、建议措施。

## 示例

- "检查我的学业预警" → 学分缺口预警（120分缺口）
- "有挂科风险吗" → 当前无挂科
