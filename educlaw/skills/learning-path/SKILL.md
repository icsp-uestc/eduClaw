# 学习路径规划

## 概述

根据目标职业方向生成个性化的分学期课程学习路径。

## 触发关键词

路径、规划、计划、安排、后端、前端、全栈、AI、人工智能

## 内置方向

| 方向 | ID |
|------|-----|
| 后端开发工程师 | backend_dev |
| 人工智能研究员 | ai_researcher |
| 全栈开发工程师 | fullstack_dev |

## 执行流程

1. 模糊匹配用户输入到内置学习路径
2. 无匹配时返回默认路径（后端开发工程师）
3. 输出路径的学期安排、课程详情

## 输出格式

路径名称、目标能力、预计学期数、总学分、描述、每学期课程列表。

## 示例

- "我想成为后端工程师" → 后端开发工程师路径
- "AI方向" → 人工智能研究员路径
