# 课程检索

## 概述

在课程数据库中按多维度条件检索课程，返回匹配度排序的结果列表。

## 触发关键词

搜索、找、查、有没有、有哪些、课程

## 支持条件

| 条件 | 示例 |
|------|------|
| 关键词 | 编程、数据库、Python |
| 课程类型 | 必修、选修、通识 |
| 难度 | 简单、中等、困难 |
| 院系 | 计算机、数学 |
| 学分 | 3学分、4学分 |

## 执行流程

1. 使用 NL2SQL 解析器从用户查询中提取关键词和过滤条件
2. 遍历课程数据库，按名称(+3)、描述(+2)、标签(+2)、条件(+5) 加权评分
3. 按得分降序排列返回匹配课程
4. 无匹配时返回建议提示

## 输出格式

返回每门课程的：名称、编号、学分、课时、类型、难度、院系、先修课程、标签、简介、教师。

## 示例

- "编程相关的课程" → 返回程序设计基础、面向对象程序设计
- "计算机专业必修课" → 过滤院系=computer + 类型=必修
