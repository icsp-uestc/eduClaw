# 课程推荐

## 概述

基于用户输入的需求描述，对所有课程进行综合评分，推荐最匹配的课程。

## 触发关键词

推荐、建议、想选、适合、怎么选

## 执行流程

1. 解析用户输入，提取关键词
2. 对课程库所有课程按名称匹配(+2)、标签匹配(+1) 评分
3. 无关键词时返回所有课程
4. 按评分降序返回前 N 门（默认5门）

## 输出格式

序号、课程名称、编号、学分、难度、简介。

## 示例

- "推荐适合初学者的编程课" → 推荐程序设计基础等
- "有什么好课" → 返回课程库前5门
