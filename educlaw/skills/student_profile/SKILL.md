# 能力画像生成

## 概述

基于学生课程成绩数据，计算 8 项能力维度的量化得分和等级，并给出个性化学习建议。同时生成雷达图可视化数据。

## 触发关键词

画像、能力、成绩、GPA、水平、分析

## 8项能力指标

| 能力 | 等级阈值 |
|------|----------|
| 编程开发 | S≥90, A≥85, B≥75, C≥60, D<60 |
| 算法设计 | 同上 |
| 系统设计 | 同上 |
| 数据库应用 | 同上 |
| 项目管理 | 同上 |
| 沟通协作 | 同上 |
| 学习能力 | 同上 |
| 创新能力 | 同上 |

## 执行流程

1. 使用内置模拟成绩数据（6门课程）
2. 按课程名称和关键词映射到8项能力维度
3. 加权计算各维度得分（按学分加权）
4. 生成GPA、能力雷达图数据、学习建议

## 输出格式

- 文本：学生摘要、能力得分表、学习建议
- 图表数据：8维度雷达图（labels, values, pointColors）
